# twisty.js

## Usage

Add to the header:

  <link href="path/to/twisty.css" rel="stylesheet" type="text/css">
  <script src="path/to/twisty.js"></script>

Automatic (applet-style):

    <twisty
      alg="R U R'"
    ></twisty>

Custom (Javascript):

    <twisty
      initialization="custom"
      alg="R U R'"
    ></twisty>

    <script>
      new TwistyPlayer(element);
    </script>
